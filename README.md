基于L大源码以及P3TERX的actions项目编译的OpenWrt.

目前包含了以下插件:
1.netdata
2.passwall2
3.广告屏蔽大师 Plus+
4.阿里云盘WebDav
5.ShadowsocksR Plus+
6.OpenClash
7.阿里DDNS
8.迅雷快鸟
9.UU游戏加速器
10.netspeedtest
