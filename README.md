### <img src="https://raw.githubusercontent.com/MartinHeinz/MartinHeinz/master/wave.gif" width="30px">基于L大源码以及P3TERX的Actions项目编译的OpenWrt.



### 目前包含了以下插件:
    ◉ netdata
    ◉ passwall2
    ◉ 广告屏蔽大师 Plus+
    ◉ 阿里云盘WebDav
    ◉ ShadowsocksR Plus+
    ◉ OpenClash
    ◉ 阿里DDNS
    ◉ 迅雷快鸟
    ◉ UU游戏加速器
    ◉ netspeedtest
